# React + Vite + Node

## Frontend

To start this project I recommend use yarn

#### Command line

- Install dependencies

`$ yarn`

- Run project

`$ yarn dev`

## Server

To start this project I recommend use yarn

#### Command line

- Access the server folder

`$ cd server`

- Install node server dependencies

`$ yarn`

- Exit server folder

`$ cd ..`

- Start server

`$ node server`
